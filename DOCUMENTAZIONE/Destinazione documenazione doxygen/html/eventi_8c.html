◆ caricaEventi()

int caricaEventi ( eventi *  eventi,
struct index *  indexs,
locali *  locali 
)

La funzione apre il file di riferimento e il file degli index, legge riga per riga il file del CSV, le analizza e le inserisce in elementi di un vettore dinamico che viene generato in base al numero di righe che vengono lette dal file. In seguito viene creato un puntatore all'index di AUTO_INCREMENT

Parametri
[in]eventi: struct contentente, tra gli altri campi, l'indirizzo all'array di struct utilizzato per effettuare operazioni concernenti agli eventi.
[in]indexs: puntatore a struct index
[in]locali: struct contentente, tra gli altri campi, l'indirizzo all'array di struct utilizzato per effettuare operazioni concernenti ai locali.
Restituisce
0 : il vettore è stato caricato con successo
-1 : impossibile aprire il file

◆ eliminaEvento()

int eliminaEvento ( eventi *  eventi,
evento *  entity,
locali *  locali 
)

Il programma esegue una ricerca di evento, e chiede una conferma per l'eliminazione. Successivamente l'evento trovato viene spostato in ultima posizione e viene fatto un realloc del vettore dinamico che così esclude l'evento da eliminare.

Parametri
[in]eventistruct contentente l'indirizzo all'array di struct (utilizzato per effettuare operazioni concernenti agli eventi) e la gradezza della stessa.
Restituisce
0: l'evento cercato non è stato trovato
1: valore che indica il successo dell'operazione di eliminazione
-2: si è deciso di non eliminare l'evento
-1: se l'array di struct concernente i locali è nullo.
